单目 > 算术 > 移位 > 关系 > 位 > 逻辑 > 条件 > 赋值 > 逗号

  • ! 优先级最高（单目）
  • && 高于 ||
  • 赋值运算符自右向左结合：a=b=c=1

// Ch5 函数

函数声明与定义

  • 函数类型由 return 决定，缺省为 int
  • 形参是局部变量，调用时分配，返回时释放
  • 数组作参数传递的是地址（可修改原数组）

变量的作用域和生存期

  • 局部变量：函数内有效，调用结束释放
  • 全局变量：函数外定义，整个文件有效
  • static 局部变量：值保留到下次调用

// Ch6 数组

一维数组

  • 定义：int a[10]; 下标从0开始
  • 初始化：int a[5]={1,2,3,4,5};
  • 部分初始化：int a[5]={1,2}; 其余为0

二维数组

  • 定义：int a[3][4]; 3行4列
  • 按行存储，先行后列
  • 省略行数：int a[][4]={{1,2},{3,4}};

字符数组与字符串

  • 字符串以 \0 结尾
  • strlen 不计 \0，sizeof 计入
  • 常用函数：strcpy、strcat、strcmp

// Ch7 指针

指针基础

  • & 取地址，* 取内容
  • int *p; p是指向int的指针
  • 指针变量本身也有地址

指针与数组

  • 数组名是首元素地址（常量指针）
  • p+i 等价于 &a[i]
  • *(p+i) 等价于 a[i]

指针与函数

  • 地址传递可修改实参
  • 函数返回指针：int *func()
  • 函数指针：int (*p)(int, int)

// Ch8 结构体

结构体定义与使用

struct Student {
    int num;
    char name[20];
    float score;
};
struct Student s1 = {101, "Li", 89.5};

结构体指针

  • p->成员 等价于 (*p).成员
  • 动态分配：p = (struct Student *)malloc(sizeof(struct Student));

// Ch1 单片机概述

基本概念

  • 单片机 = CPU + 存储器 + I/O接口（集成在一块芯片上）
  • 哈佛结构：程序存储器和数据存储器分开
  • MCS-51系列：8031/8051/8751

// Ch2 AT89S52 硬件结构

引脚功能

  • VCC(40脚)：+5V电源
  • VSS(20脚)：接地
  • XTAL1/XTAL2(19/18脚)：外接晶振
  • RST/VPD(9脚)：复位/备用电源
  • EA/VPP(31脚)：访问外部ROM允许
  • ALE/PROG(30脚)：地址锁存允许
  • PSEN(29脚)：外部ROM读选通

存储结构

  • ROM：程序存储器（片内4KB，片外可扩展64KB）
  • RAM：数据存储器（片内256B，片外64KB）
  • 片内RAM分：工作寄存器区(00-1FH)、位寻址区(20-2FH)、通用RAM(30-7FH)、SFR(80-FFH)

并行I/O口

端口特点
P0开漏输出，外扩地址/数据总线复用
P1准双向口，无第二功能
P2准双向口，外扩时作高8位地址
P3准双向口，有第二功能（串口/中断/定时器等）

复位电路

  • 上电自动复位 + 按键手动复位
  • RST保持2个机器周期高电平复位
  • 复位后：PC=0000H，P0-P3=FFH，SP=07H

// 七种寻址方式

1. 立即寻址

操作数在指令中：MOV A, #30H

2. 直接寻址

操作数为地址：MOV A, 30H（访问片内RAM 30H单元）

3. 寄存器寻址

操作数在寄存器中：MOV A, R0

4. 寄存器间接寻址

寄存器内容为地址：MOV A, @R0（R0中为地址）

5. 变址寻址

基址+变址：MOVC A, @A+DPTR

6. 相对寻址

PC+偏移量：SJMP rel（相对转移）

7. 位寻址

对位操作：MOV C, 20H（位地址20H）

// 常用指令分类

数据传送类

  • MOV - 片内RAM传送
  • MOVX - 片外RAM传送
  • MOVC - ROM查表
  • XCH - 字节交换
  • PUSH/POP - 堆栈操作

算术运算类

  • ADD/ADDC - 加法/带进位加
  • SUBB - 带借位减
  • INC/DEC - 加1/减1
  • MUL AB - 乘法（A×B→BA）
  • DIV AB - 除法（A÷B→A...B）
  • DA A - BCD调整

逻辑运算类

  • ANL - 与 / ORL - 或 / XRL - 异或
  • RL/RR/RLC/RRC - 移位
  • CPL - 取反 / CLR - 清零

控制转移类

  • LJMP/AJMP/SJMP/JMP - 无条件转移
  • JZ/JNZ - 判断A是否为0
  • CJNE - 比较不等转移
  • DJNZ - 减1不为0转移（循环常用）
  • LCALL/ACALL/RET - 子程序调用与返回

// T0/T1 核心寄存器

寄存器映射

  • TH0/TL0：定时器0高/低字节

// 四种工作方式

方式0：13位定时/计数

TH0(8位) + TL0(5位) = 13位

方式1：16位定时/计数（最常用）

TH0(8位) + TL0(8位) = 16位

计数初值计算：

  • 定时模式：X = 65536 - T×fosc/12
  • 计数模式：X = 65536 - N

方式2：8位自动重装

TL0计数，溢出时TH0自动重装，适合定时

方式3：仅T0可用

T0分为两个8位定时器，T1停止

unsigned char KeyScan(void) {
    unsigned char key = 0;
    P1 = 0x0F;
    if (P1 != 0x0F) {
        Delay_ms(20);
        if (P1 != 0x0F) {
            P1 = 0xFE;
            if (P1 != 0xFE) key = ...;
            P1 = 0xFD;
            if (P1 != 0xFD) key = ...;
        }
    }
    return key;
}
