// Ch2 数据类型与运算

基本数据类型

  • int - 整型（2/4字节）
  • float - 单精度浮点（4字节，6-7位有效数字）
  • double - 双精度浮点（8字节，15-16位有效数字）

运算符优先级（必记）

单目 > 算术 > 移位 > 关系 > 位 > 逻辑 > 条件 > 赋值 > 逗号

  • ! 优先级最高（单目）
  • && 高于 ||
  • 赋值运算符自右向左结合：a=b=c=1

  • 每个 case 后加 break，否则"穿透"

// Ch5 函数

函数声明与定义

  • 函数类型由 return 决定，缺省为 int
  • 形参是局部变量，调用时分配，返回时释放
  • 数组作参数传递的是地址（可修改原数组）

变量的作用域和生存期

  • 局部变量：函数内有效，调用结束释放
  • 全局变量：函数外定义，整个文件有效
  • static 局部变量：值保留到下次调用

// Ch6 数组

一维数组

  • 定义：int a[10]; 下标从0开始
  • 初始化：int a[5]={1,2,3,4,5};
  • 部分初始化：int a[5]={1,2}; 其余为0

二维数组

  • 定义：int a[3][4]; 3行4列
  • 按行存储，先行后列
  • 省略行数：int a[][4]={{1,2},{3,4}};

字符数组与字符串

  • 字符串以 \0 结尾
  • strlen 不计 \0，sizeof 计入
  • 常用函数：strcpy、strcat、strcmp

// Ch7 指针

指针基础

  • & 取地址，* 取内容
  • int *p; p是指向int的指针
  • 指针变量本身也有地址

指针与数组

  • 数组名是首元素地址（常量指针）
  • p+i 等价于 &a[i]
  • *(p+i) 等价于 a[i]

指针与函数

  • 地址传递可修改实参
  • 函数返回指针：int *func()
  • 函数指针：int (*p)(int, int)

// Ch8 结构体

结构体定义与使用

struct Student {
    int num;
    char name[20];
    float score;

struct Student s1 = {101, "Li", 89.5};

结构体指针

  • p->成员 等价于 (*p).成员
  • 动态分配：p = (struct Student *)malloc(sizeof(struct Student));

// Ch1 单片机概述

基本概念

  • 单片机 = CPU + 存储器 + I/O接口（集成在一块芯片上）
  • 哈佛结构：程序存储器和数据存储器分开
  • MCS-51系列：8031/8051/8751

// Ch2 AT89S52 硬件结构

引脚功能

  • VCC(40脚)：+5V电源
  • VSS(20脚)：接地
  • XTAL1/XTAL2(19/18脚)：外接晶振
  • RST/VPD(9脚)：复位/备用电源
  • EA/VPP(31脚)：访问外部ROM允许
  • ALE/PROG(30脚)：地址锁存允许
  • PSEN(29脚)：外部ROM读选通

存储结构

  • ROM：程序存储器（片内4KB，片外可扩展64KB）
  • RAM：数据存储器（片内256B，片外64KB）
  • 片内RAM分：工作寄存器区(00-1FH)、位寻址区(20-2FH)、通用RAM(30-7FH)、SFR(80-FFH)

并行I/O口

端口特点
P0开漏输出，外扩地址/数据总线复用
P1准双向口，无第二功能
P2准双向口，外扩时作高8位地址
P3准双向口，有第二功能（串口/中断/定时器等）

复位电路

  • 上电自动复位 + 按键手动复位
  • RST保持2个机器周期高电平复位
  • 复位后：PC=0000H，P0-P3=FFH，SP=07H

// 七种寻址方式

1. 立即寻址

操作数在指令中：MOV A, #30H

2. 直接寻址

操作数为地址：MOV A, 30H（访问片内RAM 30H单元）

3. 寄存器寻址

操作数在寄存器中：MOV A, R0

4. 寄存器间接寻址

寄存器内容为地址：MOV A, @R0（R0中为地址）

5. 变址寻址

基址+变址：MOVC A, @A+DPTR

6. 相对寻址

PC+偏移量：SJMP rel（相对转移）

7. 位寻址

对位操作：MOV C, 20H（位地址20H）

// 常用指令分类

数据传送类

  • MOV - 片内RAM传送
  • MOVX - 片外RAM传送
  • MOVC - ROM查表
  • XCH - 字节交换
  • PUSH/POP - 堆栈操作

算术运算类

  • ADD/ADDC - 加法/带进位加
  • SUBB - 带借位减
  • INC/DEC - 加1/减1
  • MUL AB - 乘法（A×B→BA）
  • DIV AB - 除法（A÷B→A...B）
  • DA A - BCD调整

逻辑运算类

  • ANL - 与 / ORL - 或 / XRL - 异或
  • RL/RR/RLC/RRC - 移位
  • CPL - 取反 / CLR - 清零

控制转移类

  • LJMP/AJMP/SJMP/JMP - 无条件转移
  • JZ/JNZ - 判断A是否为0
  • CJNE - 比较不等转移
  • DJNZ - 减1不为0转移（循环常用）
  • LCALL/ACALL/RET - 子程序调用与返回

// T0/T1 核心寄存器

寄存器映射

  • TMOD（89H）：工作方式控制
  • TCON（88H）：控制和标志
  • TH0/TL0：定时器0高/低字节
  • TH1/TL1：定时器1高/低字节

// 四种工作方式

方式0：13位定时/计数

TH0(8位) + TL0(5位) = 13位

方式1：16位定时/计数（最常用）

TH0(8位) + TL0(8位) = 16位

计数初值计算：

  • 定时模式：X = 65536 - T×fosc/12
  • 计数模式：X = 65536 - N

方式2：8位自动重装

TL0计数，溢出时TH0自动重装，适合定时

方式3：仅T0可用

T0分为两个8位定时器，T1停止

// TMOD寄存器

位名称功能
7GATET1门控位（1=INT1控制启停）
6C/TT1定时/计数选择（0=定时，1=计数）
5-4M1M0T1工作方式
3-0-T0同上

// 初始化步骤

定时器初始化流程

  1. 确定工作方式，写TMOD
  2. 计算计数初值，装TH0/TL0
  3. 开中断（ET0/ET1、EA）
  4. 启动定时器（TR0/TR1）
  5. 编写中断服务程序

// 五个中断源

中断源入口地址优先级
INT0（外部中断0）0003H最高
T0（定时器0）000BH↓
INT1（外部中断1）0013H↓
T1（定时器1）001BH↓
串行口0023H最低

// 中断控制寄存器

TCON寄存器（88H）

  • IT0/IT1：触发方式（0=低电平，1=下降沿）
  • IE0/IE1：中断请求标志
  • TR0/TR1：定时器启停
  • TF0/TF1：定时器溢出标志

IE寄存器（A8H）

  • EA：总中断允许
  • ET0/ET1：定时器中断允许
  • EX0/EX1：外部中断允许
  • ES：串口中断允许

IP寄存器（B8H）

  • PX0/PX1：外部中断优先级
  • PT0/PT1：定时器优先级
  • PS：串口优先级
  • 1=高优先级，0=低优先级

// 中断处理过程

  1. 中断请求（硬件置位标志）
  2. 中断响应（检查条件）
  3. 断点保护（PC自动入栈）
  4. 中断服务（执行ISR）
  5. 中断返回（RETI）

// 基本概念

通信方式

  • 串行/并行：数据位传输方式
  • 同步/异步：时钟同步方式
  • 单工/半双工/全双工：数据方向

波特率

  • 每秒传输的二进制位数
  • 常用：9600、4800、2400 bps

// SCON寄存器（98H）

位名称功能
7-6SM0-SM1工作方式选择（4种）
5SM2多机通信控制
4REN允许接收
3TB8发送第9位
2RB8接收第9位
1TI发送中断标志
0RI接收中断标志

// 四种工作方式

方式0：同步移位

8位，波特率 = fosc/12

方式1：8位UART（最常用）

10位（1起始+8数据+1停止）

波特率 = (2^SMOD/32) × T1溢出率

方式2：9位UART

11位，波特率 = (2^SMOD/64) × fosc

方式3：9位UART（可变波特率）

同方式2，波特率由T1决定

// 初始化步骤

  1. 设置SCON（工作方式）
  2. 设置PCON（SMOD波特率倍增）
  3. 设置T1（波特率发生器，方式2）
  4. 开中断（ES、EA）
  5. 启动T1（TR1）

串口初始化模板

void UART_Init(void) {
    SCON = 0x50;    // 方式1，允许接收
    PCON = 0x00;    // 不倍增
    TMOD = 0x20;    // T1方式2，自动重装
    TH1 = 0xFD;     // 9600bps (12MHz)
    TL1 = 0xFD;
    ES = 1;          // 开串口中断
    EA = 1;          // 开总中断
    TR1 = 1;         // 启动T1
}

void UART_ISR(void) interrupt 4 {
    if (RI) {
        RI = 0;
        SBUF = SBUF;  // 回显
    }
    if (TI) {
        TI = 0;
    }
}

}

延时函数模板

// 毫秒延时 (12MHz)
void Delay_ms(unsigned int ms) {
    unsigned int i, j;
    for (i = 0; i < ms; i++)
        for (j = 0; j < 120; j++);
}

// 微秒延时 (12MHz)
void Delay_us(unsigned int us) {
    while (us--);
}

GPIO操作模板

#include <reg52.h>

sbit LED = P1^0;    // LED接P1.0
sbit KEY = P3^2;    // 按键接P3.2

void main(void) {
    LED = 1;
    while (1) {
        if (KEY == 0) {
            Delay_ms(20);
            if (KEY == 0) {
                LED = ~LED;
                while (!KEY);
            }
        }
    }
}

矩阵键盘扫描模板

unsigned char KeyScan(void) {
    unsigned char key = 0;
    P1 = 0x0F;
    if (P1 != 0x0F) {
        Delay_ms(20);
        if (P1 != 0x0F) {
            P1 = 0xFE;
            if (P1 != 0xFE) key = ...;
            P1 = 0xFD;
            if (P1 != 0xFD) key = ...;
        }
    }
    return key;
}
以上模板基于12MHz晶振，实际使用时需根据晶振频率调整定时初值和延时参数。
