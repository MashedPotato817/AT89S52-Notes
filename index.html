// Ch2 数据类型与运算

基本数据类型

  • int - 整型（2/4字节）
  • float - 单精度浮点（4字节，6-7位有效数字）
  • double - 双精度浮点（8字节，15-16位有效数字）
  • char - 字符型（1字节）

运算符优先级（必记）

单目 > 算术 > 移位 > 关系 > 位 > 逻辑 > 条件 > 赋值 > 逗号

  • ! 优先级最高（单目）
  • && 高于 ||
  • 赋值运算符自右向左结合：a=b=c=1

  • 表达式必须是整型或字符型

  • 每个 case 后加 break，否则"穿透"

💻 C语言进阶（5-8章）

// Ch5 函数

函数声明与定义

  • 函数类型由 return 决定，缺省为 int
  • 形参是局部变量，调用时分配，返回时释放
  • 数组作参数传递的是地址（可修改原数组）

变量的作用域和生存期

  • 局部变量：函数内有效，调用结束释放
  • 全局变量：函数外定义，整个文件有效
  • static 局部变量：值保留到下次调用

// Ch6 数组

一维数组

  • 定义：int a[10]; 下标从0开始
  • 初始化：int a[5]={1,2,3,4,5};
  • 部分初始化：int a[5]={1,2}; 其余为0

二维数组

  • 定义：int a[3][4]; 3行4列
  • 按行存储，先行后列
  • 省略行数：int a[][4]={{1,2},{3,4}};

字符数组与字符串

  • 字符串以 \0 结尾
  • strlen 不计 \0，sizeof 计入
  • 常用函数：strcpy、strcat、strcmp

// Ch7 指针

指针基础

  • & 取地址，* 取内容
  • int *p; p是指向int的指针
  • 指针变量本身也有地址

指针与数组

  • 数组名是首元素地址（常量指针）
  • p+i 等价于 &a[i]
  • *(p+i) 等价于 a[i]

指针与函数

  • 地址传递可修改实参
  • 函数返回指针：int *func()
  • 函数指针：int (*p)(int, int)

// Ch8 结构体

结构体定义与使用

struct Student {
    int num;
    char name[20];
    float score;
};
struct Student s1 = {101, "Li", 89.5};

结构体指针

  • p->成员 等价于 (*p).成员
  • 动态分配：p = (struct Student *)malloc(sizeof(struct Student));

// Ch2 AT89S52 硬件结构

掌握片内硬件组成

AT89S52单片机内部包含：

  • CPU：8位微处理器
  • ROM：8KB Flash程序存储器
  • RAM：256B数据存储器
  • 并行I/O口：P0、P1、P2、P3四个8位口
  • 定时器/计数器：T0、T1、T2三个16位
  • 中断系统：5个中断源
  • 串行口：UART全双工串行口
出题角度：简答题"51单片机片内包含哪些功能部件？"

掌握40个引脚功能

引脚名称功能出题重点
40VCC+5V电源—
20VSS接地—
19XTAL1片内振荡器输入接晶振
18XTAL2片内振荡器输出接晶振
9RST复位信号输入高电平复位
31EA外部ROM允许EA=1用片内ROM
30ALE地址锁存允许锁存P0低8位地址
29PSEN外部ROM读选通低电平有效
出题重点：EA、ALE、PSEN三个控制引脚的功能和接法

掌握CPU主要组成：PSW和PC

程序计数器PC（16位）：

  • 存放下一条要执行指令的地址
  • 复位后PC=0000H（程序从0000H开始执行）
  • 自动加1，实现程序顺序执行

程序状态字PSW（8位）：

位名称功能重点
7CY进位标志加法进位/减法借位
6AC辅助进位BCD运算
5F0用户标志用户自定义
4RS1寄存器组选择00-11选4组
3RS0寄存器组选择
2OV溢出标志有符号数溢出
1—保留—
0P奇偶标志A中1的个数奇偶
出题角度：填空题考PSW各位的含义，选择题考RS1/RS0选择寄存器组

理解存储结构（重点：片内RAM）

ROM空间（程序存储器）：

  • 片内：8KB Flash（0000H-1FFFH）
  • 片外：可扩展64KB（0000H-FFFFH）
  • EA=1时先执行片内ROM，EA=0时只用片外

片内RAM空间（256B）：

地址区域说明
00H-1FH工作寄存器区4组×8个寄存器（R0-R7）
20H-2FH位寻址区128个位地址（00H-7FH）
30H-7FH通用RAM区数据缓冲区、堆栈
80H-FFH特殊功能寄存器SFR控制寄存器
出题重点：片内RAM四个区域的地址范围和功能，特别是位寻址区

掌握并行I/O口

端口类型第二功能外扩时作用
P0开漏输出无低8位地址/数据复用
P1准双向口无—
P2准双向口无高8位地址
P3准双向口有（串口/中断/定时等）—

P3口第二功能（常考）：

  • P3.0 - RXD（串行接收）
  • P3.1 - TXD（串行发送）
  • P3.2 - INT0（外部中断0）
  • P3.3 - INT1（外部中断1）
  • P3.4 - T0（定时器0外部输入）
  • P3.5 - T1（定时器1外部输入）
  • P3.6 - WR（外部RAM写）
  • P3.7 - RD（外部RAM读）
出题角度：填空题考P3口第二功能，选择题考哪个口有第二功能

了解时钟电路，掌握周期计算

外部时钟电路：

  • 晶振接XTAL1和XTAL2
  • 典型晶振频率：6MHz、11.0592MHz、12MHz

四种周期及计算（12MHz晶振）：

周期关系12MHz时
振荡周期= 1/fosc1/12μs
时钟周期（状态周期）= 2×振荡周期1/6μs
机器周期= 6×时钟周期 = 12×振荡周期1μs
指令周期= 1~4个机器周期1~4μs
出题重点：给定晶振频率，计算机器周期；单周期指令执行时间

掌握复位电路

复位条件：RST引脚保持2个机器周期以上的高电平

复位后SFR初值（必记）：

寄存器复位值含义
PC0000H程序从0000H开始
SP07H堆栈指针指向07H
P0-P3FFHI/O口全为高电平
PSW00H选择第0组寄存器
TMOD00H定时器方式0
TH0/TL000H定时器0清零
IE00H中断全部禁止
IP00H低优先级

复位电路类型：

  • 上电自动复位：电容充电产生高电平脉冲
  • 按键手动复位：按下按键RST接VCC
出题角度：填空题考复位条件、复位后PC/SP/P0-P3的值；画图题考复位电路

理解最小系统

最小系统组成：

  • 单片机芯片
  • 时钟电路（晶振+电容）
  • 复位电路（电阻+电容+按键）
  • 电源（VCC+5V，VSS接地）
  • EA接VCC（使用片内ROM）
